Úvod

Tato práce je studentský projekt vytvořený jako semestrální práce pro předmět Značkovací jazyky na České Zemědělské univerzitě v Praze.

Semestrální práce obsahuje XML soubor s daty kde je defonováno 50 elementů a 10 atributů s 5-ti ukázkovými záznami, vypracované XMLSchema kde nalezneme restrikce (omezení) pro každý element a atribut a XSLT soubor kde je vytvořena transformace do HTML

Práce se zabývá zpracováním objednávek pro prodej zemědělských strojů.

Metodika

Hlavní soubory této práce (XML, XMLSchema, XSLT) jsou vypracovány v Notepad++ a jejich správnost ověřena na online validátorech, například Utilities Online pro ověření validity XML dokumentu, jestli je Well formed, neboli bez chyby a jestli zároveň koresponduje s XMLSchema (XSD souborem). XSD totiž slouží jako šablona pro vkládání dat v XML, pokud tedy máme v XSD definován element jako xs:integer nelze do něv vkládat znak. Také omezuje některé elementy a atributy v zadávaní, kde se vyžaduje přesný počet čísel například u čísla účtu nebo vyplnění emailové adresy, kde adresa musí obsahovat znaky(cisla)@znaky.znaky coz je standard pro zadávání emailové adresy. Při nedodržení těchto restrikcí aplikace nová data nepřijme a vyhodí chybu.

Popis elementů a atributů

XML soubor lze rozdělit na objednávkový list neboli Objednávka, která obsahuje Objednavatele, Dodavatele a Traktor a jejich elementy a aributy.

  • Zemědělské stoje
    • Objednávka - element - data o různých objednávkách
      • Objednavkaid - atribut - rozeznání jednotlivých objednávek
      • Datum - element - datum objednavky
      • Cas - element - cas objednavky
      • Objednavatel - elemnt - data o člověku, který si objednal nějaký stroj
        • Datumnarozeni - atribut - datum narození Objednavatele
        • Titul - atribut - titul Objednavatele
        • Jmeno - element - jméno objednavatele
        • Prijmeni - element - příjmení objednavatele
        • Vek - element - vek objednavatele
        • Cislouctu - element - číslo účtu objednavatele
        • Adresa - element - adresa Objednavatele
          • Typadresy - atribut - adresa může být doručovací nebo korespondenční, adresa Objednavatele
          • Kontinet - element - jméno objednavatele
          • Stat - element - stát ve kterém bydlí objednavatel
          • Kraj - element - kraj ve kterém bydlí objednavatel
          • Mesto - element - město ve kterém bydlí objednavatel
          • Ulice - element - ulice ve které bydlí objednavatel
          • Cislo - element - číslo na kterém bydlí objednavatel
          • Psc - element - psč na které bydlí Objednavatel
      • Dodavatel - element - firma, která dodá objednané zboží
        • Nazevfirmy - element - název firmy Dodavatele
        • Url - element - url firmy Dodavatele
        • Email - element - email firmy Dodavatele
        • Telefon - element - telefon firmy Dodavatele
        • Fax - element - fax firmy Dodavatele
        • Ico - element - IČO firmy Dodavatele
      • Traktor
        • Oznaceni - atribut - Kategorie traktoru (Velké, střední, malé, speciální traktory)
        • Vyrobeno - atribut - datum vyrobení traktoru
        • Stav - atribut - stav vozidla (nový,ojetý)
        • Mnozstvi - atribut - množství objednaných traktorů s těmito parametry
        • Nazev - element - název Traktoru
        • Rada - element - řada Traktoru
        • Vyrobnicislo - element - výrobní číslo Traktoru
        • Pridavnebrzdy - element - jestli vybavení traktoru obsahuje přídavné brzdy
        • Vykon - element - výkon Traktoru
        • Obsahvalcu - element - obsah válců Traktoru
        • Vaha - element - váha Traktoru
        • Taznasila - element - tažná síla Traktoru
        • Cena - element - Cena Traktoru
        • Klicovevlastnosti - element - nadstandardní výbava traktoru
          • Typ - atribut - typ kabiny (příplatková, standardní, ..)
            • Kabina - element - vlastnosti kabiny Traktoru
            • Vyhled - element - jak velký je výhled z traktoru
            • Klimatizace - element - jestli kabina obsahuje klimatizaci
            • Zvuk - element - jestli kabina obsahuje rádio a jaké má vlastnosti
            • Ovladace - element - jestli kabina má barevně rozlišené ovladače
            • Vnejsiosvetleni - element - kolik stupňů dokáže osvítit venkovní osvětlení
            • Pocetsvetel - element - kolik světel obsahuje traktor (jsou na kambině)
          • Rizeni - element - vlastnosti řízení Traktoru
            • Nazevrizeni - atribut - název řízení
            • Druh - atribut - druh řízení (elektronicé, manuální, ...)
            • Kontrolastability - element - jakou traktor používá kontrolu stability
            • Variabilniprevod - element - muže traktor použít varabilní převod (ano, ne)
          • Prevodovka - element - vlastnosti převodovky Traktoru
            • Nazevprevodovky - atribut - název převodovky
            • Rychlosti - atribut - počet rychlostí dobředu+dozadu
            • Prepravnirychlost - element - přepravní rychlost, které traktor dosáhne
            • Otacky - element - otáčky kterých dosahuje převodovka
            • Nastavenirychlosti - element - nastavení automatické udržování rychlosti traktoru (ano, ne)
            • Prevody - element - typ prevodů, kterými převodovka disponuje

Popis transformace

Transformace je uložena v XSLT souboru a obsahuje výpis 5 záznamů. Vytvoří se tabulka kde jsou barevně odděleny sopuvisející skupiny elementů. Pomocí XSL:FOR-EACH projedu všechny záznamy a pomocí XSL:SORT je seřadím dle abecedy podle příjmení Objednavatelů.
